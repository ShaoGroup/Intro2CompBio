\section{Homework organization}
\subsection{How to submit}
The homeworks should be sumbmitted through {\it Github}\footnote{Github:https://github.com}.
\subsection{Homework format}
\begin{itemize}
\item Codes should be written by R \footnote{R:https://r-can.com}. 
\item Please follow the recommended format when you write the codes.
\item No \textbf{COPY}.
\end{itemize}
\section{Presenatation on the exercise class}
\subsection{Groups and presentation format}
\begin{itemize}
\item Three groups are classified in the first exercise class.
\item Different groups will read and present different research directions on bioinformatics. 
\item From the second to the fifth exercise class, each group will have 15 minutes to present.
\end{itemize}
\subsection{Recommended papers}\footnote{The paper list will be shown at the end of this paper.}
\begin{itemize}
\item Sequence analysis
  \begin{itemize}
  \item Multiple Sequence alignment, such as Profile-HMM.
  \item Motif Discovery, such as Gibbs Sampler.
  \item Sequence Mapping, such as Euler Graph.
  \end{itemize}
\item Systems biology and network pharmacology
  \begin{itemize}
  \item Network pharmacology, such as CIPHER.
  \item Network control
  \end{itemize}
\item Population genetics
  \begin{itemize}
  \item SNP 
  \item ENCODE
  \item GTEx
  \end{itemize}
\end{itemize}
